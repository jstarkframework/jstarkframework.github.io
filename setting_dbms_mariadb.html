이미지가 보이지 않는다면 브라우저를 최신 버전으로 변경이 필요합니다.
MARIADB의 설치 및 설정 그리고 데이터 생성 및 등록에 대해 알아보겠습니다.



1. MARIADB 설치


https://mariadb.org/ 에서 MARIADB를 다운로드하고, C:\JSTARK_INSTALL폴더에 저장합니다.
저장된 mariadb-10.0.12-win32.msi 파일을 실행합니다.




Next 버튼을 클릭합니다.




라이선스 동의를 체크하고 Next 버튼을 클릭합니다.




Location을 C:\DEV\MariaDB 로 변경하고 Next 버튼을 클릭합니다.




root password를 "jstark"로 입력하고 UTF8을 기본설정으로 체크하고 Next 버튼을 클릭합니다.




Next 버튼을 클릭합니다.




Next 버튼을 클릭합니다.




Install 버튼을 클릭합니다.




MARIADB 설치가 종료되었습니다. Finish 버튼을 클릭합니다.




바탕화면의 바로가기 아이콘이나 C:\Program Files (x86)\Common Files\MariaDBShared\HeidiSQL 폴더의 heidisql.exe를 실행합니다.




신규를 선택하고 루트 폴더 내에 세션 생성을 선택합니다.




세션관리자에서 암호에 설치 시 입력한 "jstark"를 입력하고 열기 버튼을 클릭합니다.




MARIADB의 설치와 준비가 종료되었습니다.



2. 데이터베이스 생성


데이터베이스 생성 파일은 C:\DEV\jstarkProject\jstarkbizsuite\WEB-INF\resource\setup\jstark_database\mariadb에 존재하며 해당 폴더로 이동하여 확인합니다.




HeidiSQL를 실행하고 암호를 jstark 을 입력하고 열기 버튼을 클릭합니다.




SQL 파일 불러오기를 통해 C:\DEV\jstarkProject\jstarkbizsuite\WEB-INF\resource\setup\jstark_database\mariadb 폴더로 이동한 후 01_jstark_create.sql을 선택한 뒤 열기 버튼을 클릭합니다.




SQL 실행 아이콘을 클릭합니다.




데이터베이스 생성이 종료되었습니다.



3. 데이터 등록


데이터 등록을 위해 DB 연결 설정을 MARIADB에 맞게 변경합니다. 아래와 같이 jstarkbizsuite/WEB-INF/src/jstark/framework/conf/jstark.application.properties 의 jdcp connection 설정의 변경이 필요합니다.



/jstarkbizsuite/WEB-INF/resource/setup/conf/jdbc.example.properties 파일의 내용을 참조하여 아래와 같이 jstark.application.properties의 jdcp 연결 정보를 변경합니다.




C:\DEV\jstarkEclipse로 이동한 후 eclipse.exe로 eclipse를 실행합니다.




아래와 같이 jstarkbuzsuite project가 추가되어 있고, Properties를 클릭합니다.




Resource에서 Text file encoding이 UTF-8로 되어있는지 확인하고 MS949로 설정되어 있으면 UTF-8로 변경하고 OK 버튼을 클릭합니다.




데이터 등록을 위해 c:\DEV\tomcat6\bin의 startup.bat을 클릭하여 tomcat을 실행시킵니다. tomcat은 80포트로 설정되어 있기 때문에 정상적으로 서비스가 실행되지 않으면 tomcat6/conf 의 설정파일을 확인하세요.




C:\DEV\jstarkProject\jstarkbizsuite\WEB-INF\resource\setup\jstark_database 폴더의 jstark_data_insert.jsp 파일을 복사한 뒤 c:\DEV\jstarkPorject\jstarkbizsuite 폴더에 붙여 넣습니다.






웹 브라우저에 아래와 같이 주소를 입력합니다.
http://localhost/jstark_data_insert.jsp



데이터 등록이 종료되었습니다. jstark_data_insert.jsp 파일을 삭제합니다.




설치 및 설정이 종료되었습니다.
