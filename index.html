JSTARK FRAMEWORK
웹 어플리케이션 개발에 필요한 ALL-IN-ONE 개발 환경을 제공하는 자바 프레임워크(Java Framework).
- 신속한 설치(20분)
- 짧은 학습 시간(2시간)
- 높은 생산성/성능/안정성/보안
- 15년 이상의 운영을 통한 성능 검증

웹 어플리케이션 개발을 위한 최고의 선택!!
JSTARK PLATFORM
글로벌 비즈니스 시스템을 위한 플랫폼으로
Client, User, Menu, Language, Code, Tree, Theme, Security, oAuth, Rest Api, Chat, Screen Share 등의 기능을 제공합니다.

2014년 미래창조과학부 주최, 제8회 공개SW 개발자대회(OSS WORLD CHALLENGE 2014)
일반 자유 부문 동상 수상 (JSTARK PLATFORM)
JSTARK CONSOLE
Log Monitoring를 통해 수집된 성능 로그의 분석으로 시스템 최적화가 가능합니다.
실시간 상태 모니터링(APM : Application Performance Management)과 환경 모니터링(ARM : Application Runtime Environment) 기능을 지원합니다.
JSTARK SCHEDULE
시스템 내에서 보다 쉽게 스케줄 설정이 가능하며,
설정 값에 따라 자동으로 해당 프로그램 실행하고 실시간으로 로그를 확인할 수 있습니다.
JSTARK SDK/EXAMPLE
코드 자동 생성 및 코드 생성 지원 도구와 다양한 샘플 예제/템플릿으로 학습 및 개발 시간을 단축하고,
동일 품질의 개발로 향후 유지보수 시간의 단축이 가능합니다.
