ORACLE의 설치 및 설정 그리고 데이터 생성 및 등록에 대해 알아보겠습니다..



1. Oracle 설치


http://www.oracle.com에서 Oracle Database Express Edition 11g 를 다운로드하고 C:\JSTARK_INSTALL 폴더에 저장합니다. 저장된 OracleXE112_Win32.zip 파일을 압축 해제 한 후 Disk1\setup.exe를 실행합니다.




License 동의를 위해 I accept the terms in the license agreement를 선택 한 후 Next 버튼을 클릭합니다.




설치경로 변경을 위해 Browse 버튼을 클릭합니다.




설치경로를 아래와 같이 c:\DEV\oraclexe 폴더로 지정한 후 확인 버튼을 클릭합니다.




설치 경로를 확인 한 후 Next 버튼을 클릭합니다.




아래는 Oracle SYS 계정과 SYSTEM 계정의 Password를 설정하는 화면으로 패스워드를 “jstark”로 입력합니다. 개발 및 교육을 위한 Oracle 이기 때문에 예시와 같은 password를 사용해도 무방하지만 실제에서는 보안을 위해 다른 password를 사용하세요. 만약 암호를 변경한 경우 tomcat의 conf에서 변경한 암호로 연결 설정값을 변경해야 합니다.




Install 버튼을 클릭합니다.




설치 도중 아래와 같은 오류 메시지가 나타날 경우 사용에는 문제가 없으니 확인 버튼을 클릭하여 다음으로 넘어갑니다.




Finish 버튼을 클릭합니다.




Oracle 설치가 종료되었습니다.



3. 데이터베이스 생성


데이터베이스 생성 파일은 C:\DEV\jstarkProject\jstarkbizsuite\WEB-INF\resource\setup\jstark_database\oracle에 존재하며 CMD를 이용하여 해당 폴더로 이동합니다.




CMD 창에서 oracle 폴더로 이동을 위해 아래와 같이 cd 명령어를 입력한 뒤 sysdba 계정으로 sqlplus에 접속합니다.
C:\> cd C:\DEV\jstarkProject\jstarkbizsuite\WEB-INF\resource\setup\jstark_database\oracle
C:\DEV\jstarkProject\jstarkbizsuite\WEB-INF\resource\setup\jstark_database\oracle> sqlplus / as sysdba



XDB 삭제, Tablespace, Temp Tablespace 생성, User 생성, User 권한설정 등을 위해 아래와 같이 입력합니다.
SQL> @jstark_create





데이터베이스 생성이 종료되었습니다.



4. 데이터 등록


데이터 등록을 위해 c:\DEV\tomcat6\bin의 startup.bat을 클릭하여 tomcat을 실행시킵니다. tomcat은 80포트로 설정되어 있기 때문에 정상적으로 서비스가 실행되지 않으면 tomcat6/conf 의 설정파일을 확인하세요.




C:\DEV\jstarkProject\jstarkbizsuite\WEB-INF\resource\setup\jstark_database 폴더의 jstark_data_insert.jsp 파일을 복사한 뒤 C:\DEV\jstarkPorject\jstarkbizsuite 폴더에 붙여 넣습니다.






웹 브라우저에 아래와 같이 주소를 입력합니다.
http://localhost/jstark_data_insert.jsp



데이터 등록이 종료되었습니다. jstark_data_insert.jsp 파일을 삭제합니다.




설치 및 설정이 종료되었습니다.
