1. WebLogic Download


WebLogic 다운로드를 위해 oracle web site ( http://www.oracle.com/us/downloads/index.html )에 접속합니다.

Downloads - Middleware Oracle Fusion Middleware (including Oracle WebLogic Server)을 클릭합니다.




사용 동의를 위해 Accept License Agreement 를 선택하고 Windows 32 installer including OEPE(1.8GB)를 선택한 후 Download File 버튼을 클릭합니다.






다운로드를 위해서는 오라클 계정이 필요하며, 오라클 계정이 없는 경우 계정 생성을 통해 계정을 생성한 후 로그인을 합니다.




아래의 경로에 설치 파일을 다운로드 합니다.

C:\DEV\JSTARK_SETUP\oepe-12.1.1.1-kepler-installer-win32.exe

추가자료 (Jar Classpath)

Weblogic 도 Tomcat과 마찬가지로 jstark_external_lib의 API들을 WAS의 lib에 복사하여야 합니다.
Weblogic의 user_project lib 경로는 버전에 따라 다르지만 Weblogic이 설치되어 있는 폴더
예) c:\DEV\Oracle\Middleware\Oracle_Home\user_projects\domains\base_domain\lib 
에 복사하며, 이 경로를 Classpath에 등록하기 위해 -Dweblogic.ext.dirs 옵션으로 Weblogic을 실행합니다.
jstark_external_lib의 경우 대부분 WAS의 lib로 추가하는 것이 좋지만,
EL과 JSTL의 경우 Application lib 폴더인 WEB-INF/lib에 복사하여야 오류가 나지 않습니다.
