1 Apache Tomcat 다운로드


Tomcat을 설치하기 위해 http://tomcat.apache.org/ 웹사이트에 접속합니다.
Tomcat의 경우 6, 7, 8 모두 지원하기 때문에 어떤 버전을 사용해도 상관없습니다. 여기서는 Tomcat 6.0으로 진행해 보겠습니다. 아래와 같이 좌측 Download의 Tomcat 6.0을 클릭합니다.




Tomcat의 경우 버전에 따라 Minimum Java Version이 있기 때문에 아래와 같이 Which version에서 JDK 버전과 Tomcat 버전을 참고하여 설치하여야 합니다.




아래 Binary Distributions에서 Windows 버전을 설치합니다. 우리는 Oracle 을 32비트 버전을 받았기 때문에 Tomcat 및 JDK도 동일한 32비트를 다운로드 합니다. Tomcat의 경우 일반 실행(32bit Windows zip)과 서비스 등록(32-bit/64-bit Windows Service Installer) 버전이 있습니다. 실제 운영에서는 서비스 등록 버전을 설치하는 것이 좋으며, 개발 및 학습을 위해서 우리는 32-bit Windows zip 버전을 클릭합니다.




설치파일은 아래의 경로에 저장합니다.

C:\DEV\JSTARK_SETUP\apache-tomcat-6.0.39-windows-x86.zip



2. Tomcat 설치


Tomcat의 설치는 C:\DEV\JSTARK_SETUP\apache-tomcat-6.0.39-windows-x86.zip 파일을 압축 해제 한 후 폴더명을 tomcat으로 변경 한 후 C:\DEV\tomcat6 폴더로 이동합니다.





3 Tomcat 설정


WAS(Web Application Server)인 Tomcat에 Library 파일 추가를 합니다. JSTARK_SOURCE \ jstark_external_lib 폴더의 모든 파일을 복사(Ctrl+C)합니다.




복사한 파일들을 Tomcat의 lib 폴더(C:\DEV\tomcat6\lib)에 붙여넣기를 합니다.




Tomcat에 Library 파일 등록이 완료되었습니다.

다음으로 Tomcat 설정을 위해 C:\DEV\tomcat6\conf 폴더로 이동합니다.



context.xml 파일을 메모장으로 열고, 아래와 같이 수정합니다.
<Context reloadable="true" privileged="true">




tomcat-users.xml 파일을 아래와 같이 수정합니다.
<user username="jstark" password="jstark" roles="manager-gui"/>



server.xml 파일을 아래와 같이 수정합니다.

Port와 URIEncoding을 변경합니다.
<Connector port="80" protocol="HTTP/1.1" URIEncoding="utf-8" connectionTimeout="20000" redirectPort="8443" />



Context root를 등록합니다.
<Context path="" docBase="C:\DEV\jstarkProject\jstarkbizsuite" debug="0" reloadable="true" privileged="true"/>



Tomcat 설정 변경이 완료되었습니다.

추가자료 (윈도우 서비스로 등록하기)


OS마다 다르지만 윈도우의 경우 startup.bat로 Tomcat을 실행하면 사용자 세션이 항상 로그인 되어 있어야 하고
서버가 재구동 되는 경우 수동으로 다시 startup.bat를 실행시켜야 한다.
따라서 자동으로 서버가 올라오면 Tomcat이 실행되게 하기 위해서는 서비스 형태의 Tomcat을 설치하는 방법이 있고, 
Tomcat을 서비스에 등록하는 방법이 있다. Tomcat을 서비스로 등록하는 방법은 아래와 같다.

1. CMD 에서 아래와 같은 명령어로 tomcat의 bin 폴더로 이동한다.
cd C:\DEV\tomcat6\bin

2. service.bat의 명령으로 서비스에 등록한다.
service.bat install jstark

3. 작업관리자-서비스 에서 시작유형을 자동으로 변경하고 실행을 시킨다. 


서비스에서 삭제는 아래 명령어로 서비스에서 삭제한다.
service.bat uninstall jstark
