1. POSTGRESQL 설치


http://www.enterprisedb.com/products-services-training/pgdownload# 에서 POSTGRESQL를 다운로드하고, C:\JSTARK_INSTALL폴더에 저장합니다.
저장된 postgresql-9.3.5-1-windows.exe 파일을 실행합니다.




Next 버튼을 클릭합니다.




Installation Directory를 C:\DEV\PostgreSQL\9.3으로 변경 후 Next 버튼을 클릭합니다.




Next 버튼을 클릭합니다.




password를 "jstark"로 입력하고 Next 버튼을 클릭합니다.




Next 버튼을 클릭합니다.




Next 버튼을 클릭합니다.




Next 버튼을 클릭합니다.




Stack Builder may be used to download and install additional tools~ 를 체크 해제하고 Finish 버튼을 클릭합니다.




계정/Database 생성을 위해 SQL Shell (psql)을 실행합니다.




엔터키를 입력합니다.




엔터키를 입력합니다.




엔터키를 입력합니다.




엔터키를 입력합니다.




password에 jstark 를 입력하고 엔터키를 입력합니다. 입력한 password는 보이지 않으니 주의하여 입력합니다.




postgres 계정 생성이 완료되었습니다.




paAdmin III를 실행합니다.




PostgreSQL 9.3을 선택 후 마우스 오른쪽 버튼을 클릭하여 Connect를 선택합니다.




Store password를 체크하고 password로 jstark를 입력한 후 OK 버튼을 클릭합니다.




OK 버튼을 클릭합니다.




PostgreSQL 9.3 - Database - postgres 를 클릭합니다.



POSTGRESQL의 설치와 준비가 종료되었습니다.



2. 데이터베이스 생성


데이터베이스 생성 파일은 C:\DEV\jstarkProject\jstarkbizsuite\WEB-INF\resource\setup\jstark_database\postgresql에 존재하며 해당 폴더로 이동하여 확인합니다.




pgAdmin III에서 SQL 아이콘을 클릭합니다.




Query에서 Open File을 선택한 후 C:\DEV\jstarkProject\jstarkbizsuite\WEB-INF\resource\setup\jstark_database\postgresql 폴더로 이동하여 01_jstark_create.sql을 선택한 뒤 열기 버튼을 클릭합니다.




Execute Query 아이콘을 클릭합니다.




데이터베이스 생성이 종료되었습니다.

데이터 등록을 위해 DB 연결 설정을 POSTGRESQL에 맞게 변경합니다. 아래와 같이 jstarkbizsuite/WEB-INF/src/jstark/framework/conf/jstark.application.properties 의 jdcp connection 설정의 변경이 필요합니다.




/jstarkbizsuite/WEB-INF/resource/setup/conf/jdbc.example.properties 파일의 내용을 참조하여 아래와 같이 jstark.application.properties의 jdcp 연결 정보를 변경합니다.





C:\DEV\jstarkEclipse로 이동한 후 eclipse.exe로 eclipse를 실행합니다.




아래와 같이 jstarkbuzsuite project가 추가되어 있고, Properties를 클릭합니다.




Resource에서 Text file encoding이 UTF-8로 되어있는지 확인하고 MS949로 설정되어 있으면 UTF-8로 변경하고 OK 버튼을 클릭합니다.




데이터 등록을 위해 c:\DEV\tomcat6\bin의 startup.bat을 클릭하여 tomcat을 실행시킵니다. tomcat은 80포트로 설정되어 있기 때문에 정상적으로 서비스가 실행되지 않으면 tomcat6/conf 의 설정파일을 확인하세요.
