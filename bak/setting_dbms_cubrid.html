1. CUBRID 설치


http://www.cubrid.com에서 CUBRID-Windows-x64-9.3.0.0206.exe 와 CUBRIDManager-2014-build-0458-windows-i386.exe를 다운로드하고 C:\JSTARK_INSTALL 폴더에 저장합니다. 설치를 위해 먼저 CUBRID-Windows-x64-9.3.0.0206.exe 실행합니다.

다음 버튼을 클릭합니다.




설치 경로를 C:\DEV\CUBRID로 변경하고 다음 버튼을 클릭합니다.

2. 데이터베이스 생성


데이터베이스 생성 파일은 C:\DEV\jstarkProject\jstarkbizsuite\WEB-INF\resource\setup\jstark_database\cubrid에 존재합니다.




파일 - SQL 파일 열기를 클릭합니다.




C:\DEV\jstarkProject\jstarkbizsuite\WEB-INF\resource\setup\jstark_database\cubrid 경로의 01_jstark_create.sql을 선택하고, 문자집합을 UTF-8로 지정 후 확인 버튼을 클릭합니다.




실행 아이콘을 클릭합니다.




데이터베이스 생성이 종료되었습니다.



3. 데이터 등록


데이터 등록을 위해 DB 연결 설정을 CUBRID에 맞게 변경합니다. 아래와 같이 jstarkbizsuite/WEB-INF/src/jstark/framework/conf/jstark.application.properties 의 jdcp connection 설정의 변경이 필요합니다.




/jstarkbizsuite/WEB-INF/resource/setup/conf/jdbc.example.properties 파일의 내용을 참조하여 아래와 같이 jstark.application.properties의 jdcp 연결 정보를 변경합니다.




C:\DEV\jstarkEclipse로 이동한 후 eclipse.exe로 eclipse를 실행합니다.




아래와 같이 jstarkbuzsuite project가 추가되어 있고, Properties를 클릭합니다.




Resource에서 Text file encoding이 UTF-8로 되어있는지 확인하고 MS949로 설정되어 있으면 UTF-8로 변경하고 OK 버튼을 클릭합니다.




데이터 등록을 위해 c:\DEV\tomcat6\bin의 startup.bat을 클릭하여 tomcat을 실행시킵니다. tomcat은 80포트로 설정되어 있기 때문에 정상적으로 서비스가 실행되지 않으면 tomcat6/conf 의 설정파일을 확인하세요.

웹 브라우저에 아래와 같이 주소를 입력합니다.
http://localhost/jstark_data_insert.jsp



데이터 등록이 종료되었습니다. jstark_data_insert.jsp 파일을 삭제합니다.
