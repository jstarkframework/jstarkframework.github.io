JSTARKBIZSUITE의 FULL VERSION 설치에 대해 알아보겠습니다.



1. FULL VERSION 설치

JSTARK BizSuite의 빠른 설치를 위해 설정이 모두 되어 있는 Full Version으로 설치를 진행해 보겠습니다. JSTARK Full Version은 32bit로 구성되어 있고, 아래와 같이 구성되며, DBMS와 나눔글꼴은 해당 사이트에서 다운받으셔야 합니다.

    - java 1.6
    - tomcat 6
    - eclipse 4.3 kepler

    - DBMS(Oracle, Mssql, MariaDB, Cubrid, PostgreSql)
    - 나눔글꼴 (http://dev.naver.com/projects/nanumfont/download/)

http://jstark.org 에서 Full Version을 다운받고 c:\JSTARK_INSTALL 폴더에 아래와 같이 저장합니다.




JSTARK_FULL.ZIP을 압축 해제합니다.
압축 파일에 포함된 이클립스에 긴 파일명이 존재하여 압축 해제시 오류가 발생하면 7Zip과 같은 프로그램을 이용하여 압축을 하제하면 오류 없이 해제가 됩니다.




이제 아래와 같은 순서로 설치 및 설정을 진행하겠습니다.

    1. 나눔글꼴 설치
    2. DEV 복사
    3. 시스템 환경변수 설정
    4. DBMS 설치/설정/데이터 등록
    5. 시스템 접속



1. 나눔글꼴(나눔고딕코딩) 설치


http://dev.naver.com/projects/nanumfont/download/ 에서 나눔글꼴(나눔고딕코딩)을 다운로드하고,
c:\JSTARK_INSTALL 폴더에 저장합니다. 저장된 파일 중 NanumGothicCoding_Setup-2.0.exe 파일을 실행합니다. 설치 버튼을 클릭합니다.




마침 버튼을 클릭합니다.




설치가 종료되었습니다.



2. DEV 복사


C:\JSTARK_INSTALL\JSTARK_FULL\DEV 폴더를 C:\DEV로 복사합니다.

    - java
    - jstarkEclipse
    - jstarkProject
    - tomcat6







3. 시스템 환경변수 설정


JDK 사용을 위해 시스템 환경변수에 JAVA_HOME과 Path에 추가가 필요합니다. 아래와 같이 탐색기에서 “컴퓨터” 항목을 마우스 오른쪽 버튼의 클릭을 통해 속성 항목을 클릭합니다.




시스템 창에서 고급 시스템 설정을 클릭한 후 고급 탭 하단의 환경변수 버튼을 클릭합니다.




JAVA_HOME 변수 추가를 위해 새로 만들기 버튼을 클릭한 후 변수이름과 변수값을 아래와 같이 입력한 후 확인 버튼을 클릭합니다.
간혹 OS에 설치 폴더가 다를 수 있으니 JAVA 설치시 지정한 폴더명을 확인해야 합니다.
변수 이름 : JAVA_HOME
변수 값 : C:\DEV\java\jdk1.6.0_20



다음으로 기존에 등록되어있는 Path 항목을 선택한 후 편집 버튼을 클릭하고, 아래와 같이 변수 값의 맨 앞부분에 다음을 추가한 후 확인 버튼을 클릭합니다. 각각의 변수는 ; 로 구분됩니다.
C:\DEV\java\jdk1.6.0_20\bin;



설정이 완료되면 DOS command 창을 실행시켜 아래와 같이 자바 버전을 확인합니다.
C:\> java -version




4. DBMS 설치/설정/데이터 등록


Oracle, Mssql, Mariadb, Cubrid, PostgreSql 중 원하는 DBMS를 선택하여 설치/설정/데이터 등록을 진행하세요. DBMS SETUP GUIDE를 참고하세요.



5. 시스템 접속


웹 브라우저에 아래와 같이 주소를 입력합니다.
http://localhost/
로그인 화면이 나타납니다.




초기 아이디 비밀번호는 jstark / 111111 이며 로그인 후 패스워드 변경 화면으로 이동합니다.
